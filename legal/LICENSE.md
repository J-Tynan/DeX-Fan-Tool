# Licensing Notes

The primary repository license is defined in the top-level `LICENSE.md`.

## Covered By The Repository License

- Original source code, scripts, and documentation authored for this repository.
- Project-authored examples, diagrams, and notes unless a file says otherwise.

## Not Automatically Covered

- Vendor firmware, firmware images, or extracted binary blobs.
- Raw USB captures or logs that may contain vendor-derived or sensitive material.
- Third-party utilities, archives, or reference material stored in `third_party/` or `tools/`.
- Reproduced vendor documentation, screenshots, or protocol dumps copied from external sources.

## How To Use This Folder

Use `legal/` for licensing analysis, third-party notice planning, and documentation of any exceptions or future release constraints.

When there is any doubt about whether an artifact is safe to redistribute, treat the artifact as restricted until it has been reviewed against `legal/DISTRIBUTION_POLICY.md`.
