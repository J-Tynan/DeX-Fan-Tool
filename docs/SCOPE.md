# Scope and Non-Goals

## In scope
- USB enumeration and safe IN probing from Android.
- Logging and analysis of USB responses.
- Implementing a small Android UI to trigger discovered commands and map phone temperature to fan speed.
- Documenting procedures, safety checks, and test results.
- Hardware bypass (MCU + MOSFET) as a documented fallback.

## Non-goals
- Releasing tools that intentionally circumvent vendor DRM or protections for distribution.
- Modifying or redistributing Samsung One UI or system images.
- Commercialization without legal review.
- Any action that knowingly endangers users or property.
